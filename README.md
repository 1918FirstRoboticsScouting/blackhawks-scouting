# Blackhawks Scouting

## Overview

Blackhawks Scouting is an advanced, web-based, and open source scouting app made by FRC Team 2834 (The Bionic Blackhawks).

Powered by [Vue.js](https://vuejs.org) and [The Blue Alliance](https://thebluealliance.com).
